# Python PMS5003
Python program to read data from PLANTOWER PMS5003

Uses passive mode of sensor, so you can specify delay between following measurements in data_delay
Prints PM1, PM2.5 and PM10 values in ug/m^3 for both standard particle and atmospheric conditions as well as numbers of particles in #/cm^3

Sensor datasheet: http://www.aqmd.gov/docs/default-source/aq-spec/resources-page/plantower-pms5003-manual_v2-3.pdf
